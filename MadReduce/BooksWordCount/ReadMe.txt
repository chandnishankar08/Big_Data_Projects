ProjectName: Word Count without stop words
Tools used: Java, hadoop, MapReduce

Website referred for Stop Words: http://www.lextek.com/manuals/onix/stopwords1.html

Command to run program:
hadoop jar WordCount.jar WordCount.WordCount.WordCount3 <input directory path> <output directory path>

Example:
hadoop jar WordCount.jar WordCount.WordCount.WordCount3 assignment1/textbook/ assignment1/output_textbook
